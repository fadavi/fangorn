# Fangorn

## Requiremenst
Python 3.10.x

## How to run
Directly using python:
```bash
$ python3 -mfangorn
```

Or using make:
```bash
$ make pvp # Orderus vs. Beast
$ make tdm # Team Deathmatch
```
